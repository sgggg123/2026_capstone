## 0. 우리가 답하려는 질문

**"전쟁이나 위기가 터지면, 비트코인은 금처럼 가격이 오를까?"**

옛날에는 사람들이 위기 때 금을 샀다. 가치가 안전하니까. 그래서 금 = "안전자산". 비트코인은 디지털 금이라고 불리니까, 위기가 와도 가격이 안정적이거나 오히려 오를까? 그게 우리가 보는 거다.

## 2. 우리가 본 3가지

### ① Event Study — "위기 직후 BTC 가격 어땠어?"
> "위기 발생일 ±N일 동안 비트코인 가격이 평소보다 얼마나 더 오르거나 떨어졌는가?"

**비유**: 시험 보기 전 학생의 평소 점수가 80점인데, 시험 당일 90점 받으면 +10점이 "초과 성과". 이걸 매일 더하면 누적 초과 성과(CAR).

**판정**: CAR가 양수이고 통계적으로 의미 있으면 → 안전자산.

### ② GARCH — "위기가 BTC 변동성을 키우나?"
> "전쟁 뉴스 같은 외부 충격이 비트코인 가격의 출렁임을 더 크게 만드는가?"

**비유**: 평소 잔잔한 호수 (=변동성 작음). 돌을 던지면 (=뉴스) 파도가 친다 (=변동성 커짐). 어떤 돌이 가장 큰 파도를 만드는지 보는 거.

**모델**: σ²(t) = ω + α·ε²(t-1) + β·σ²(t-1) + γ·X(t)
- σ² = 변동성
- α = 어제 충격이 얼마나 영향
- β = 어제 변동성이 얼마나 지속
- γ·X = 외부 변수(전쟁지수)가 영향
- **γ > 0 이고 유의하면 → 위기지수가 변동성을 키움**

### ③ Quantile Regression — "주가가 폭락할 때 BTC 는 반대로 가나?"
> "S&P500 이 5% 하위로 폭락하는 날, BTC 는 같이 떨어지나? 반대로 가나?"

**비유**: 친구 무리에서 누가 진짜 안전한 사람인지 보려면, **위기 상황 (싸움이 벌어진 순간)** 을 봐야 한다. 평소엔 다 비슷해 보이지만, 위기 때 도와주는 친구가 진짜 안전.

**판정**: 하위 5~10% 분위(폭락 구간)에서 BTC 가 SP500 과 반대 방향(β < 0) → 안전자산.

## 3. 팀원이 했던 분석의 문제 4가지 (간단히)

### 🔴 문제 1: GARCH 코드가 망가져 있었음
**비유**: 자동차 엔진이 시동 걸리는데 RPM 게이지가 0 에 붙어있다. 운전은 되지만 모든 계기가 의심스러움.

GARCH 의 핵심 모수 (alpha, omega)가 경계값에 붙어서 의미 없는 결과. **t-stat = 5000** 같은 비정상 수치.

**우리 수정**: alpha 의 가능 범위를 0.001 이상으로 강제. 8번 다른 시작점에서 다시 돌려서 가장 좋은 결과 채택.

### 🔴 문제 2: Event Study 윈도우를 멋대로 줄임
**비유**: "내가 한 달 동안 다이어트를 했어!" 라고 하다가 "사실 일주일이야"로 줄이면 결과 의미 달라짐.

팀원은 처음 ±25 일로 시작해서 → ±17 → 결국 ±3 일로 줄임. 결과가 윈도우 길이에 따라 달라지는지 검증 안 함.

**우리 수정**: ±3, ±5, ±10, ±17 4개 윈도우로 동시에 분석. 결과가 일관된 이벤트 = robust, 윈도우에 따라 바뀌는 이벤트 = 의심.

### 🔴 문제 3: BTC 주말 가격을 평균 처리
**비유**: 수능 점수 매길 때, "주말에 본 모의고사 점수도 1/3 씩 평균해서 월요일 점수에 합치자"고 하면? 평균 효과로 점수 변동성이 작아져 보임. 진짜 실력 측정 X.

팀원 코드: 월요일 BTC 수익률 = (금 + 토 + 일) / 3.

**우리 검증**: yfinance 에서 BTC 가격 다시 받아 표준 방식 (그냥 종가 차이) 으로 다시 계산.
- 결과: **팀원 방식이 BTC 변동성을 15.5% 축소시킴**.
- 즉 모든 분석이 "BTC 가 사실보다 차분하다"고 보고 있었던 것.

### 🔴 문제 4: 표준오차가 부정확
GARCH 결과의 p-value(=확률) 가 신뢰성 부족. 진짜 효과가 있는데 "효과 없음" 으로 잘못 판단.

**우리 수정**: arch 라이브러리 (산업 표준) 도 같이 돌려서 비교. 두 방법 결과 일치 / 불일치 확인.

## 4. 우리가 새로 발견한 것

### 🎯 발견 1: VIX(시장공포)가 진짜 강한 변수
팀원 결과: VIX γ = ?, p = 0.82 → 비유의 (영향 없음).

**우리 수정 결과**: VIX γ = +7.56, **p = 0.001** → 매우 유의.

해석: **위기 자체보다 시장의 공포 심리가 BTC 변동성을 더 강하게 결정**한다.

> "전쟁 뉴스로 직접 BTC 변동이 커지는 게 아니라, 그 뉴스 때문에 시장 전체가 두려워하면 BTC 도 흔들린다."

### 🎯 발견 2: 이스라엘-하마스 결과는 의심스러움
팀원 결과: ±17 일 윈도우에서 BTC CAR = +33%, p = 0.012 → Safe Haven.

**우리 검증**: 윈도우 ±10 → "Marginal", ±5 → "Non-Sig", ±3 → "Non-Sig".

> 윈도우 길이가 결과를 결정하면, 그 결과는 **분석자가 어떤 길이를 선택했느냐의 산물**. 진짜 효과 아님.

### 🎯 발견 3: 결국 BTC = 안전자산 X
6개 이벤트 중 단 1개 (이-하마스 ±17 만) 에서 Safe Haven 분류. 그것도 robust 하지 않음.

**결론**: 비트코인은 위기 때 금처럼 행동하지 않는다.
