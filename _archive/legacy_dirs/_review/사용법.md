## 3. 분석 스크립트 실행

### 한 번에 모두 실행
```bash
cd /mnt/d/project/2026_capstone
python3 _review/scripts/01_garch_fixed.py
python3 _review/scripts/02_event_study_sensitivity.py
python3 _review/scripts/03_btc_logic_ablation.py
```

각 스크립트는 `_review/results/` 에 CSV 자동 생성. 콘솔에 진단 메시지 출력.

### 개별 실행

| 스크립트 | 작업 | 출력 | 소요시간 |
|---|---|---|---|
| `01_garch_fixed.py` | GARCH 두 방법 비교 (arch lib + 커스텀 FIXED) | 5 CSV | ~30 초 |
| `02_event_study_sensitivity.py` | Event Study 윈도우 ±3/5/10/17 | 3 CSV | ~3 분 (block bootstrap 5000 회) |
| `03_btc_logic_ablation.py` | BTC 비거래일 로직 ablation | 2 CSV | ~10 초 (yfinance 다운로드 포함) |

## 4. Streamlit 대시보드 실행

```bash
cd /mnt/d/project/2026_capstone
python3 -m streamlit run app.py
```

브라우저가 자동 열리고 `http://localhost:8501` 접속됨.

### 페이지 구성 (사이드바에서 선택)

| 페이지 | 내용 |
|---|---|
| 🏠 개요 & 결론 | 6 이벤트 카드, 핵심 결론, 측정값 요약 |
| 🔬 Event Study (윈도우 sensitivity) | 4 윈도우 × 6 이벤트 × 5 자산 분석. 자산 선택 가능. CAR 히트맵, 분류 표, p-value 표, robustness 진단 |
| 📊 GARCH 변동성 (수정판) | A/B 방법 선택. 모델 비교 표, γ 계수 + 95% CI, 경계 수렴 검증 |
| 🧪 BTC 비거래일 로직 ablation | 분포 통계 표, 분포 시각화, 일별 차이 시계열, 진단 결론 |
| 📋 수정 사항 요약 | 4건 수정 + 14건 미수정 리스트 |
| 📚 방법론·수식 | LaTeX 수식, 모델 정의, 추정·검정 방법 |

### 종료
브라우저 닫고 터미널에서 `Ctrl+C`.

### 외부 공유
1. **로컬에서만 보여주기**: 위 명령 그대로
2. **같은 네트워크 (강의실 등)**: `python3 -m streamlit run app.py --server.address 0.0.0.0` → 다른 PC 에서 `http://<내IP>:8501` 접속
3. **GitHub 에 올리고 Streamlit Cloud 무료 배포** (선택): `streamlit.io/cloud` 에서 repo 연동. `requirements.txt` 가 root 에 있어야 함.

```python
es = pd.read_csv('_review/results/event_study_sensitivity.csv')

# BTC 만, 윈도우 ±10 일
btc10 = es[(es['asset']=='BTC') & (es['window']==10)]
print(btc10)

# GARCH FIXED 결과
g = pd.read_csv('_review/results/garch_custom_fixed.csv')
print(g[g['p_value']<0.05])  # 유의 변수만
```

## 7. 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| `ModuleNotFoundError: arch` | pip 미설치 | `pip install --user --break-system-packages arch` |
| streamlit 안 열림 | port 8501 이미 사용중 | `--server.port 8502` 옵션 |
| yfinance 다운로드 실패 | rate limit | 5분 후 재시도, 또는 `Edit_mj/returns.csv` 사용 |
| 한글 폰트 깨짐 (그래프) | Linux/WSL fonts-nanum 미설치 | `apt install fonts-nanum` |
| `KeyError: 'event_study_sensitivity.csv'` | 스크립트 미실행 | `python3 _review/scripts/02_*.py` 먼저 실행 |
| 커스텀 MLE 수렴 실패 | 데이터 표본 부족 | 정상 (210행은 GARCH 권장 300+ 미달) |
